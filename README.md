# Academia JJB - Demo

Aplicación web demo para una academia de Jiu-Jitsu Brasileño.

## Funcionalidades

- Registro de alumnos
- Login de usuarios
- Visualización de turnos
- Reserva de turnos
- Cancelación de reservas
- Check-in de llegada
- Panel del profesor
- Vista de mis reservas

## Tecnologías usadas

- Node.js
- Express
- HTML
- CSS
- JavaScript
